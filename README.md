# DSA Forge - AI-Powered DSA Learning Platform

**DSA Forge** is a full-stack web application that helps developers master Data Structures and Algorithms through AI-generated personalized roadmaps and structured problem-solving.

## 🚀 Features

- **AI-Powered Roadmap Generation**: Get personalized learning paths generated by GPT-4o based on your skill level, time commitment, and focus areas
- **Progress Tracking**: Monitor your learning journey with detailed analytics and completion statistics
- **Curated Problem Library**: Access 24+ hand-picked DSA problems categorized by topic and difficulty
- **Interactive Dashboard**: Visual representation of your progress with charts and metrics
- **User Authentication**: Simple user management with skill level tracking
- **Responsive Design**: Beautiful, modern UI built with Tailwind CSS and Shadcn UI

## 🛠️ Tech Stack

### Frontend
- **Framework**: React 19 (JavaScript)
- **Routing**: React Router DOM
- **UI Components**: Shadcn UI (Radix UI primitives)
- **Styling**: Tailwind CSS
- **HTTP Client**: Axios
- **Notifications**: Sonner

### Backend
- **Framework**: FastAPI (Python)
- **Database**: SQLite (SQL-based, easily switchable to PostgreSQL)
- **ORM**: SQLAlchemy
- **AI Integration**: OpenAI GPT-4o via Emergent Integrations
- **Authentication**: Simple user-based auth (easily extendable)

### Database Schema

```sql
-- Users table
users (id, username, email, skill_level, created_at, updated_at)

-- Roadmaps table
roadmaps (id, user_id, title, description, skill_level, duration_weeks, topics, generated_at)

-- Problems table
problems (id, title, difficulty, topic, description, solution_link, tags, created_at)

-- User Progress table
user_progress (id, user_id, problem_id, status, completed_at, notes, created_at)

-- Roadmap Problems (Junction table)
roadmap_problems (id, roadmap_id, problem_id, week_number, day_number, order_index, created_at)
```

## 📡 API Endpoints

### User Management
- `POST /api/users` - Create a new user
- `GET /api/users` - List all users
- `GET /api/users/{user_id}` - Get user details

### Roadmap Generation
- `POST /api/generate-roadmap` - Generate AI-powered roadmap
- `GET /api/roadmaps/{roadmap_id}` - Get roadmap details
- `GET /api/users/{user_id}/roadmaps` - Get all user roadmaps

### Problems
- `GET /api/problems` - List all problems (filters: topic, difficulty)
- `GET /api/problems/{problem_id}` - Get problem details
- `POST /api/problems` - Create a new problem

### Progress Tracking
- `POST /api/progress` - Update problem progress
- `GET /api/users/{user_id}/progress` - Get user progress
- `GET /api/users/{user_id}/dashboard-stats` - Get dashboard statistics

## 🚦 Running the Application

### Check Service Status
```bash

```

### View Logs
```bash
# Backend logs
tail -f /var/log/supervisor/backend.*.log

# Frontend logs
tail -f /var/log/supervisor/frontend.*.log
```

## 🔄 Switching to PostgreSQL

To switch from SQLite to PostgreSQL, use the provided `database_schema.sql` file and update the `DATABASE_URL` in `.env`.
